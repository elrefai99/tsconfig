# @mohamed-elrefai/tsconfigs

## Feature

- Best practice.
- One-line of tsconfig.

## Requirement

- Typescript 4.8+.
- Node 18.

## Usage

### Install

```bash
npm i @mohamed-elrefai/tsconfigs -D
```

### Config `tsconfig.json`

```json
{
  "extends": "@mohamed-elrefai/tsconfigs"
}
```
